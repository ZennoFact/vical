# vical - "vi" like "cal"culator
A calculator that runs in the Windows Command Prompt.

## Supported Operating Systems
- Windows 11

## Installation
Download zip only

## How to Use
- vical: Launches the application
- vical expression: Performs calculations in the CLI

## Help

- General
  - q: Quit app (Input mode)
  - ?: Open help
  - Esc: Close help / cancel history selection  
- History
  - j or Down: Move to newer entry
  - k or Up: Move to older entry  
  - Enter: Insert selected result 
-Commands
  - :add -> Add Sequence Mode
  - :sub -> Sub Sequence Mode
  - :mul -> Mul Sequence Mode
  - :div -> Div Sequence Mode
  - :calc -> Nomal Mode
- Sequence   
  - c: Clear sequence accumulator
